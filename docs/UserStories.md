# User Stories

## Epic 1: Account Onboarding

- As a new user, I want to create an account so that I can participate in the SkillSwap marketplace.
- As a new user, I want to verify my email so that the platform can trust my account contact information.
- As a returning user, I want to log in securely so that I can access my profile, messages, and bookings.
- As a user, I want to reset my password so that I can recover access if I forget it.
- As a user, I want to control my profile visibility so that I can decide whether others can discover me.

## Epic 2: Profile Creation

- As a user, I want to create a profile with a photo, bio, location preference, languages, and timezone so that others can understand who I am.
- As a user, I want to see profile completion guidance so that I know what information improves my chances of finding a match.
- As a mentor, I want to display credentials and examples of my work so that learners can evaluate my skill level.
- As a learner, I want to state my learning goals so that mentors can decide whether they can help me.

## Epic 3: Skill Listings

- As a mentor, I want to list skills I can teach so that learners can find me.
- As a mentor, I want to define session duration, format, level, and price so that expectations are clear before requests arrive.
- As a learner, I want to list skills I want to learn so that the platform can recommend relevant mentors or swaps.
- As a user, I want to pause a skill listing so that I can stop receiving requests when I am unavailable.
- As an administrator, I want to manage skill categories so that marketplace content remains organized.

## Epic 4: Discovery and Matching

- As a learner, I want to search for skills by category, price, availability, rating, and format so that I can find suitable mentors quickly.
- As a user, I want to see recommended matches so that I can discover people who complement my offered and wanted skills.
- As a user, I want to understand why someone is recommended so that I can decide whether the match is useful.
- As a user, I want to save interesting profiles so that I can contact them later.

## Epic 5: Requests and Scheduling

- As a learner, I want to send a session request with preferred times and a message so that the mentor can respond with enough context.
- As a mentor, I want to accept, decline, or propose changes to a request so that I can manage my schedule.
- As a user, I want available times shown in my timezone so that I do not accidentally book the wrong time.
- As a user, I want to reschedule a confirmed session so that real-life conflicts can be handled without canceling.
- As a user, I want session reminders so that I do not miss confirmed bookings.

## Epic 6: Skill Swaps

- As a user, I want to offer one of my skills in exchange for learning another so that I can participate without paying money.
- As a user, I want both sides of a swap to be clearly defined so that expectations are balanced.
- As a user, I want to confirm completion of a swap session so that reviews and reputation can be updated.

## Epic 7: Paid Sessions

- As a learner, I want to see the total cost before payment so that I can make an informed booking decision.
- As a mentor, I want to receive payment for paid sessions so that I can monetize my expertise.
- As a user, I want cancellation and refund rules to be visible so that I understand the consequences of changing plans.
- As an administrator, I want to review payment disputes so that I can help resolve issues fairly.

## Epic 8: Messaging

- As a user, I want to message another user after a valid marketplace interaction so that we can clarify session details.
- As a user, I want booking updates to appear in the conversation so that I can follow the history in one place.
- As a user, I want to report inappropriate messages so that the platform can respond to unsafe behavior.
- As a blocked user, I should not be able to continue messaging the person who blocked me.

## Epic 9: Reviews and Reputation

- As a learner, I want to review a mentor after a completed session so that future learners can make better decisions.
- As a mentor, I want to review a learner or swap partner so that the marketplace can reward respectful participation.
- As a user, I want reputation indicators to reflect verified completed sessions so that ratings are harder to manipulate.
- As an administrator, I want to remove abusive or fraudulent reviews so that reputation remains trustworthy.

## Epic 10: Notifications

- As a user, I want to receive notifications for requests, messages, booking changes, and reminders so that I can respond promptly.
- As a user, I want to manage notification preferences so that I only receive useful non-critical alerts.
- As a user, I want important security and booking emails to always be delivered so that I do not miss critical information.

## Epic 11: Safety and Moderation

- As a user, I want to block another user so that they cannot contact me.
- As a user, I want to report profiles, skills, messages, reviews, or bookings so that unsafe behavior can be investigated.
- As an administrator, I want a moderation queue so that I can review reports efficiently.
- As support staff, I want an audit trail of actions so that decisions can be reviewed later.

## Epic 12: Administration and Operations

- As an administrator, I want to view platform activity metrics so that I can monitor marketplace health.
- As an administrator, I want to manage users and content so that I can enforce platform rules.
- As support staff, I want to inspect booking history so that I can help users resolve disputes.
- As an administrator, I want role-based access so that staff only see tools required for their responsibilities.

## Prioritization

### Must Have

- Account registration and authentication.
- Profile management.
- Skills offered and wanted.
- Search and discovery.
- Swap requests and booking lifecycle.
- Messaging for active interactions.
- Reviews after completed sessions.
- Reporting and blocking.
- Basic administration.

### Should Have

- Paid sessions.
- Recommendation-based matching.
- Advanced availability management.
- Notification preferences.
- Mentor analytics.

### Could Have

- Calendar integrations.
- Saved searches.
- Endorsements.
- Learning plans.
- Group sessions.

### Won't Have Initially

- Native mobile apps.
- Built-in video conferencing.
- Enterprise integrations.
- AI tutoring automation.
